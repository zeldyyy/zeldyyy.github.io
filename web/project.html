Aplikasi Web Full Stack

Pengembangan full stack adalah proses pengembangan frontend dan backend aplikasi. Setiap aplikasi memiliki komponen frontend (yang menghadap pengguna) dan komponen backend (basis data dan logika). Frontend berisi antarmuka pengguna dan kode yang terkait interaksi pengguna dengan aplikasi.
